1.注册

接口地址：

http://104.237.154.88:8080/baiyun/index.php/Log/Api_UserRegister
参数值备注
phonenumner(电话号码)必需
email(邮箱)必需
password(密码)必需，两次MD5加密
verify(验证码)必需
typeJSON/XML可选

验证码接口：

http://104.237.154.88:8080/baiyun/index.php/Public/Verify

接口返回值：

resultCodeMsg
00001手机号码已经注册
00002注册失败
00003注册成功
00004验证码错误

2.登陆

接口地址：

http://104.237.154.88:8080/baiyun/index.php/Log/Api_UserLogin
参数值备注
phonenumber(电话号码)必选
password（密码）必选，两次MD5加密
typeJSON/XML可选

接口返回值：

resultCodeMsg
0000用户不存在
0001登陆成功
0002密码错误

3.车源

接口地址：

全部车源：

http://104.237.154.88:8080/baiyun/index.php/Cars/Api_Cars

单条车源：

http://104.237.154.88:8080/baiyun/index.php/Cars/Api_Cars
参数备注
id必选，要请求的车源id,在全部车源中有对应id
typeJSON/XML

条件查询：

http://104.237.154.88:8080/baiyun/index.php/Cars/Api_CarSearch
参数值备注
fromprovince(出发地省)可选,若不选，请传“不限”
fromcity(出发地市)可选,若不选，请传“不限”
toprovince(目的地省)可选,若不选，请传“不限”
tocity(目的地市)可选,若不选，请传“不限”
typeJSON/XML可选

地图接口：

http://104.237.154.88:8080/baiyun/index.php/Map/map
参数值备注
from(起始地)车源请求中有返回对应地图url,使用即可
to(目的地)车源请求中有返回对应地图url,使用即可
